# Articles

### All vignettes

- [Getting Started with
  hurdlebb](https://joonho112.github.io/hurdlebb/articles/01-getting-started.md):

  A hands-on introduction to fitting Hurdle Beta-Binomial models with
  hurdlebb: explore data, specify a model, run MCMC diagnostics,
  posterior predictive checks, and cross-validation — all using
  pre-computed results.

- [Survey Design
  Corrections](https://joonho112.github.io/hurdlebb/articles/02-survey-design.md):

  Incorporating complex survey weights into Hurdle Beta-Binomial models:
  weighted pseudo-posteriors, sandwich standard errors, design effect
  ratios, and Wald confidence intervals.

- [State-Varying
  Coefficients](https://joonho112.github.io/hurdlebb/articles/03-state-varying-coefficients.md):

  Extending the Hurdle Beta-Binomial to capture state-level
  heterogeneity through random intercepts and slopes, with
  interpretation of variance components, cross-margin correlations, and
  model comparison via LOO-CV.

- [Policy
  Moderators](https://joonho112.github.io/hurdlebb/articles/04-policy-moderators.md):

  Cross-level interactions: explaining state variation in Hurdle
  Beta-Binomial coefficients using policy variables — CCDF market rates,
  tiered reimbursement, and infant/toddler add-ons.

- [Marginal
  Effects](https://joonho112.github.io/hurdlebb/articles/05-marginal-effects.md):

  Average marginal effect decomposition for the Hurdle Beta-Binomial:
  separating extensive-margin (participation) and intensive-margin
  (enrolment share) contributions, with the poverty reversal
  probability.
