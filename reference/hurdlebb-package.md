# hurdlebb: Hurdle Beta-Binomial Models for Bounded Discrete Proportions

Fits two-part hurdle beta-binomial models for bounded discrete
proportions with structural zeros. Supports survey-weighted
pseudo-posterior inference, state-varying coefficients, sandwich
standard errors, and marginal effect decomposition via 'CmdStan'.
Designed for analysis of complex survey data where a proportion of units
report zero counts.
