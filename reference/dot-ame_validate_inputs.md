# Validate inputs for the ame() function

Validate inputs for the ame() function

## Usage

``` r
.ame_validate_inputs(fit, cholesky, sandwich, level, n_draws)
```

## Arguments

- fit:

  An object to validate as hbb_fit.

- cholesky:

  An optional hbb_cholesky object.

- sandwich:

  An optional hbb_sandwich object.

- level:

  Confidence level.

- n_draws:

  Optional integer.

## Value

Invisible NULL; aborts on failure.
